About Driver Logging App

Driver Logging App


Using The App

This app is intended to have a numerical keypad login, once past this point you proceed to a menu page. At the menu page you are presented with an option to input a new entry to Log Driver Distance, view Driver Fuel Consumption Logs or view the Analysis Log Graph.

  • Log Driver Distance: Allows drivers to input their name, Distance Driven, and Fuel Used.
  • Fuel Consumption Log: Allows administrators and accounting personnell the ability to view driver inputs.
  • Log Graph: Allows administrators to view driver log data in a graph for analytics.
  • Logout: returns user to the passcode screen and ends session.
